# OxideC

A modern Objective-C inspired dynamic object runtime in Rust.

## Overview

OxideC provides a safe, high-performance dynamic runtime with:
- Dynamic dispatch with late binding and method caching
- Memory safety through safe abstractions over unsafe internals
- C ABI compatibility for FFI
- Runtime reflection and introspection

## Status

**Version**: 0.1.0-alpha (Phase 1: Foundation)
